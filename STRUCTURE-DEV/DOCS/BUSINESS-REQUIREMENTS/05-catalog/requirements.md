## 1. Описание раздела

Управление товарной базой: создание, редактирование, удаление товаров, привязка к маркетплейсам через артикулы/баркоды, загрузка фото.

---

## 2. Целевая аудитория

| Роль | Действия доступны |
|------|------------------|
| OWNER / ADMIN | Полный CRUD |
| MANAGER | Редактирование, без удаления |
| STAFF | Только просмотр |

## 4. Пользовательские сценарии (User Stories)

**US-01:** Как MANAGER, я хочу добавить товар с фото и SKU, чтобы начать отслеживать остатки.

**US-02:** Как ADMIN, я хочу импортировать товары из WB Excel, чтобы не вносить вручную.

**US-03:** Как OWNER, я хочу удалить товар, чтобы убрать его из каталога (soft delete).

---

## 5. Функциональные требования

### Обязательные (Must Have)

- [ ] FR-01: Создание товара (name, SKU, фото, себестоимость)
- [ ] FR-02: Редактирование товара
- [ ] FR-03: Soft delete (товар не удаляется из истории)
- [ ] FR-04: Привязка к WB через wbBarcode (nmID)
- [ ] FR-05: Привязка к Ozon через ozonSku
- [ ] FR-06: Импорт из WB Excel
- [ ] FR-07: Фото товара → S3

### Желательные (Should Have)

- [ ] FR-10: Поиск и фильтрация по каталогу
- [ ] FR-11: Лимит товаров по тарифу

## 7. Бизнес-правила

- SKU уникален внутри Tenant.
- Без wbBarcode — синхронизация с WB не работает.
- Удалённые товары (deletedAt != null) не синхронизируются.

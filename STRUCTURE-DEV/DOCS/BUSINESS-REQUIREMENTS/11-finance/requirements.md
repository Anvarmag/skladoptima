## 4. Пользовательские сценарии (User Stories)

**US-01:** Как OWNER, я хочу видеть маржинальность каждого товара, чтобы принимать решения о ценообразовании.

**US-02:** Как ADMIN, я хочу задать себестоимость товара, чтобы система рассчитала прибыль.

---

## 5. Функциональные требования

### Обязательные (Must Have)

- [ ] FR-01: Себестоимость на товар (costPrice)
- [ ] FR-02: Расчёт маржи = revenue - costPrice - marketplace fees
- [ ] FR-03: ROI = прибыль / себестоимость * 100%
- [ ] FR-04: Настройка налоговой системы (УСН 6% / УСН 15% / ОСНО / ИП без НДС)

### Желательные (Should Have)

- [ ] FR-10: Учёт комиссий маркетплейса
- [ ] FR-11: Финансовые снэпшоты по периодам
